# Animal Cube Dash

A kid-friendly 2D platformer for Apple Watch built with SpriteKit + SwiftUI.

By Ahaana Sistla!

## About

Animal Cube Dash is a fun side-scrolling platformer where you guide adorable animal cubes through obstacle-filled levels. Jump over spikes, duck under flying knives, avoid quicksand, and reach the treasure chest to complete each level.

## Features

- **33 Animal Skins** across 5 rarity tiers:
  - **Common** (7 skins) - Cat, Dog, Bunny, Frog, Chick, Hamster, Duck
  - **Rare** (7 skins) - Panda, Fox, Owl, Penguin, Koala, Raccoon, Deer
  - **Ultra Rare** (6 skins) - Tiger, Wolf, Lion, Dolphin, Red Panda, Chameleon
  - **Epic** (7 skins) - Dragon, Phoenix, Unicorn, Shark, Golden Tiger, Arctic Fox, Snowy Owl
  - **Legendary** (11 skins) - Sanrio characters: Hello Kitty, My Melody, Cinnamoroll, Kuromi, Pompompurin, Keroppi, Pochacco, Badtz-Maru, Tuxedo Sam, Lala, Kiki

- **Obstacles** - Spikes, flying knives, quicksand pits, moving platforms
- **Duck Mechanic** - Swipe down to duck under flying knives
- **Skin Unlock System** - Earn skins by completing levels and achieving goals
- **Unlock Animations** - Gift box reveal for rare+ skins, spin animation for common skins
- **Brag System** - Share achievements with friends via Game Center or Messages
- **Game Center** - Leaderboards and friend challenges
- **Procedural Levels** - Each level is generated with increasing difficulty

## Unlock Requirements

| Rarity | Requirement |
|--------|------------|
| Common | Complete 10 levels |
| Rare | Finish a level with 2+ lives remaining |
| Ultra Rare | Finish a level with 2+ lives remaining |
| Epic | Finish a level with 2+ lives remaining |
| Legendary | Pass 5 consecutive levels without losing any lives |

## Controls

- **Tap** - Jump
- **Swipe Down** - Duck (to avoid flying knives)
- **Digital Crown** - Control scroll speed

## Requirements

- watchOS 10.0+
- Xcode 15+
- Swift 5.9+

## Building

1. Open `AnimalCubeDash.xcodeproj` in Xcode
2. Select the "AnimalCubeDash Watch App" scheme
3. Build and run on an Apple Watch or simulator
